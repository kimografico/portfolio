# Proposal: Graphic Design Section

## Objective

Showcase and explain graphic design projects by Kimo, highlighting process, results, and learnings, organized by category for clarity and depth.

## Target Audience

- Potential clients
- Recruiters
- Creative industry peers

## Unique Value

- Professional, visually appealing presentation
- Context and creative process for each project
- Technical details and final results
- Category-based navigation (branding, stationery, posters, multimedia, illustration, packaging, others)
- Advanced filtering and search in "all" view

## Initial Scope

- Category galleries (one per type)
- Individual project pages (not modals)
- All-projects view with filters and search
- Data managed in JSON files per category
- Lightbox for images

## Inspiration

- Leading designer portfolios
- Behance, Dribbble, and reference sites
